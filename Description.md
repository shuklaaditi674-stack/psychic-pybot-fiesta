# 🤖 Psychic PyBot Fiesta

> A fun, friendly, rule-based Python chatbot with personality, typing effects, and smart intent matching.

---

## ✨ About the Project

**Psychic PyBot Fiesta** is a beginner-friendly Python chatbot that responds to user inputs using keyword-based logic.  
It feels interactive with a typing animation and multiple response styles for natural conversation.

---

## 🚀 Features

- 💬 Smart keyword-based chat system  
- 🤖 Friendly PyBot personality  
- ⌨️ Typing effect for realistic replies  
- 🎭 Multiple response variations (not repetitive)  
- 🧠 Handles greetings, jokes, feelings, coding talk, etc.  
- ⏰ Dynamic time response  
- ❌ Graceful exit handling  

---

## 🛠️ Tech Stack

- Python 3.x  
- Built-in modules only:
  - `random`
  - `time`
  - `datetime`
